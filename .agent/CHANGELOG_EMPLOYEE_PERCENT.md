# Зміни в модальному вікні "Співробітники"

## Дата: 2025-12-26

## Опис змін

Модифіковано модальне вікно для управління співробітниками (`modal-content-all_other_bases`) для додавання підтримки двох процентів: **Процент роботи** та **Процент з запчастин**.

---

## Основні зміни

### 1. Додано поле "Процент з запчастин"
- **Відображення**: Поле "Процент з запчастин" відображається поруч з полем "Процент роботи" (50/50).
- **Динамічна видимість**: Поле "Процент з запчастин" з'являється **ТІЛЬКИ** якщо для співробітника обрано роль **"Приймальник"**.
- Якщо обрано будь-яку іншу роль (Слюсар, Адміністратор тощо), поле приховується, а "Процент роботи" займає всю ширину (100%).

### 2. Збереження даних (CRUD)
- При створенні нового співробітника або редагуванні існуючого зберігаються обидва значення:
  - `ПроцентРоботи`
  - `ПроцентЗапчастин`
- Дані зберігаються в таблицю `slyusars` у стовпчик `data`.

### 3. Технічна реалізація
- **Файли**:
  - `slusar.ts` (логіка інпутів, перемикання видимості, завантаження даних)
  - `vikno_pidtverdchennay_inchi_bazu.ts` (логіка збереження в базу)
  - `_dodatu_inchi_bazu_danux.scss` (стилі CSS)
- **Функція перемикання**: `updatePercentInputsVisibility(role)` керує відображенням другого інпута.

---

## Інструкція для перевірки

1. Відкрийте модальне вікно "Додати" -> "Співробітники".
2. Змініть "Доступ" на "Приймальник" -> з'явиться друге поле "Процент з запчастин".
3. Змініть "Доступ" на "Слюсар" -> друге поле зникне, перше розтягнеться.
4. Введіть значення в обидва поля (для Приймальника) і збережіть.
5. Виберіть цього співробітника в пошуку -> мають підтягнутися обидва значення.
